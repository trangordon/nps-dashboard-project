# US National Parks Dashboard

A simple interactive dashboard built with explore U.S. National Parks activity data.

## Files

* app.py – Runs the dashboard
* data.py – Fetches and processes data from the NPS API

## Setup

1. Install dependencies:
   pip install streamlit plotly pandas requests
   
## Run the App

streamlit run app.py

## Features

* Filter parks by state
* Interactive map of park locations
* Top activities chart
* Click a park to see details
* View park data in a table

## Data Source

National Park Service API: [https://developer.nps.gov/](https://developer.nps.gov/)
